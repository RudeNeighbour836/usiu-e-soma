What is Cryptography

Cryptography provides for secure communication in the presence of malicious third-parties—known as adversaries. Encryption uses an algorithm and a key to transform an input (i.e., plaintext) into an encrypted output
What is the difference between network security and cryptography? In simple words, it refers to keeping data secure.

Difference between Cryptography and Cyber Security.

Cyber Security


It is a process of keeping networks, devices, programs, data secret and safe from damage or unauthorized access.

Cryptography

It is a process of keeping information secret and safe simply by converting it into unintelligible information and vice-versa.
